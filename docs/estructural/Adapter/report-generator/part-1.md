# Ejercicio — Adapter

## 🎯 Paso 1: La versión simple

* Estás construyendo un sistema de análisis de pagos para un e-commerce.
* El sistema tiene una clase `ReportGenerator` que genera reportes de ventas.

* `ReportGenerator` trabaja con objetos del tipo `Pago`
* `ReportGenerator` tiene un método generar reporte, donde imprime el id y el monto del pago con su moneda.
---

## Tu tarea

1. Creá las clases `Pago` y `ReportGenerator`.
2. Escribí un cliente que cree una lista de 3 pagos y genere el reporte.
3. Hacé el diagrama de clases.
